## 项目介绍

abeille-manager，一个后端管理系统，使用 Vue + Vuetify。

### 目标功能
- [x] 登录注册
- [ ] 修改密码
- [ ] 第三方登录(github、码云)
- [ ] 个人中心
- [ ] 消息提醒
- [x] 在线编辑文章
- [x] 文章详情展示 
- [x] 管理博客
- [x] 自定义主题
- [ ] 评论回复系统
- [ ] 点赞、收藏
- [ ] 分类、标签、归档展示
- [x] 照片墙、关于我
- [ ] 全文检索/站内搜索
