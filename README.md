## 项目介绍

一个使用 Vue3、 Tailwindcss 的网站管理系统，致力于促进管理系统界面更好的体验。
包含帖子管理、影视资料管理、权限管理。

### 目标功能
- [x] 登录注册
- [x] 权限管理
- [x] 个人中心
- [ ] 消息提醒
- [ ] 自定义主题
- [ ] 全文检索/站内搜索
- [ ] 动画
- [x] 响应式
